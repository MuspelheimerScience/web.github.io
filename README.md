# Página oficial Hub Latino Americano de Bioimágen a través de Hardware de Acceso Libre 
LIBRE_hub es una red de capacitación para hardware de bioimágenes de código abierto en América Latina con el objetivo de entrenar a los investigadores regionales a través de talleres prácticos, seminarios, redes y recursos en línea adaptados a las necesidades locales y reeditados en los idiomas locales
